# ▲●▼◆ 한선씨 (HanSeon-C)

**4상균형3진 ISA729 프로그래밍 언어**

한글로 프로그래밍하는 4세대 온톨로직 컴퓨팅 체계.

```
함수 팩토리얼(n) {
    만약 (n < 2) { 반환 1 }
    반환 n * 팩토리얼(n - 1)
}
출력(팩토리얼(5))    // → 120
```

## 언어

```
변수 x = 42              // 변수
상수 PI = 3              // 상수
함수 더하기(a, b) {       // 함수
    반환 a + b
}
만약 (x > 0) {            // 분기
    출력("양수")
} 아니면 {
    출력("음수")
}
동안 (x > 0) {            // 루프
    x = x - 1
}
변수 상태 = 티             // 균형3진: 티(+1) 옴(0) 타(-1)
```
