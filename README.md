# ▲●▼◆ 크라우니 — 4세대 온톨로직 균형3진 체계

## 설치

```bash
tar xzf 크라우니-v0.3.1.tar.gz
cd 크라우니
./install.sh
```

## 사용법

```bash
# OS 터미널 시작
크라우니 $CROWNY_HOME/체계/쉘.한선씨

# REPL (대화형)
크라우니 대화

# 시험 실행
크라우니

# 한선씨 파일 실행
크라우니 프로그램.한선씨
```

## 구성

| 계층 | 설명 | 코드 |
|------|------|------|
| 기초 | 균형3진법, 트릿, 트릿단어, 티옴타음 | Rust |
| 온톨로지 | 27슬롯 셀, Claim DB, Layer 0~4 | Rust |
| 크라우니어 | 스택 VM, 60+ 명령어 | Rust |
| 삼진VM | ISA729 레지스터 VM, 44 옵코드 | Rust |
| 한선씨 | 한글 프로그래밍 언어 | Rust 점화기 |
| 셀프호스팅 | 한선씨로 한선씨 컴파일러 | 한선씨 |
| 크라우니OS | 터미널/쉘 | 한선씨 |

## 3진 논리 (1급 시민)

```한선씨
변수 a = 티          // +1 (확정)
변수 b = 타          // -1 (오해)
출력값(a ~~ b)       // 0 (합의: 불일치 → 중립)

판정(조건) {
    양이면 { 출력값("확정") }
    중이면 { 출력값("보류") }
    음이면 { 출력값("부정") }
}
```
