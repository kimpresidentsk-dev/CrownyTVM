# ▲■▼ CrownyOS v0.32.0 — 완전 자립 부트스트랩

## Rust/C 없이 한선씨가 자기 자신으로 영구 동작하는 체계

```
Rust 3,254줄 (점화기, 1회 사용 후 제거 가능)
한선씨 3,113줄 (49%, 21개 모듈) — VM + 컴파일러 + OS + 기계어 생성 + C변환
```

## 자립 구조

```
한선기계어.han → [1회 생성] → 기계어 시드 (ELF 345바이트)
                                │
                                ↓ .hanc 바이트코드 실행
                    ┌───────────────────────────┐
                    │ 한선씨 전체 스택           │
                    │  렉서 → 파서 → 코드젠     │
                    │  VM → 커널 → TritFS → 셸  │
                    │  크라우니셀 → 비트연산     │
                    │  메모리 할당기 → 시드재생  │
                    └───────────────────────────┘
```

Rust 점화기를 마지막으로 1회 사용하여 시드를 생성하면, 이후 한선씨가 자기 자신의 시드를 재생성 가능. gcc 경로(한선C변환.han)도 이미 검증됨.

## 사용법

```bash
crowny shell                              # ★ 대화형 CrownyOS 터미널
crowny run examples/stage7_bootstrap.han  # 자립 부트스트랩 검증
crowny run examples/stage6_no_rust.han    # Rust-free C 경로
crowny run examples/stage5_native.han     # 기계어 직접 생성
crowny run examples/stage4_demo.han       # CrownyOS 부팅
crowny run examples/stage3_demo.han       # 셀프호스팅 컴파일러
crowny test                               # 78개 테스트
```
